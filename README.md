## ğŸ“Š Allocation Techniques Implemented

### ğŸ”¹ 1. Contiguous Allocation
- Allocates consecutive disk blocks  
- Very fast access time  
- May cause external fragmentation  
- Difficult file expansion  

### ğŸ”¹ 2. Linked Allocation
- Allocates non-consecutive blocks  
- Blocks are logically linked  
- Avoids external fragmentation  
- Higher access time due to traversal  

### ğŸ”¹ 3. Indexed Allocation
- Uses an index block  
- Stores addresses of allocated blocks  
- Efficient random access  
- Requires extra index block
